# ProxyTools
#### Возвращает список из 300 прокси:
``` py
>>>from ProxyTools import *
>>>get_free_proxies()
>>>['161.202.226.194:80', .....,'5.252.161.48:8080', '52.251.88.114:80']
```
#### Принимает  список прокси
#### Возвращает сессию с рандомным прокси
``` py
>>>from ProxyTools import *
>>>proxies = ['161.202.226.194:80', .....,'5.252.161.48:8080', '52.251.88.114:80']
>>>get_session(proxies)
>>><requests.sessions.Session object at 0x000002201F1FFFC8>
```
#### Принимает прокси ip:port
#### Возвращает bool True - прокси работают, False - прокси не работают
``` py
>>>from ProxyTools import *
>>>proxies = ['161.202.226.194:80', .....,'5.252.161.48:8080', '52.251.88.114:80']
>>>is_bad_proxy('161.202.226.194:80')
>>>True
```
